# Roadmap

## v0.1 (current)

- CLI: `init`, `seed-tags`, `import-change`
- Default tag pack (PRD + planning + tasks format + TDD + archive)
- `import-change` project selection (prompt or `--project`)
- `import-change` idempotent imports (skip duplicates by default)
- `new-change` creates OpenSpec change + skeleton files
- `plan-change` creates a planning task in Vibe Kanban

## v0.2

- Optional: expand more `@tags` into imported task bodies
- Import “update existing” mode (sync task descriptions)

## v0.3

- Custom OpenSpec schema/templates for VKFlow
- GitHub Issue/PR linking in imported tasks
- Optional: generate a “planning task” card automatically
